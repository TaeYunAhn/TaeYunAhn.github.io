# node:46998 [DEP0148] 오류 수정방법

평소 npm run deploy 를 하면 중간에 이런 오류가 떴었다.

![스크린샷 2022-01-01 20.13.34.png](node%2046998%20%5BDEP0148%5D%20%E1%84%8B%E1%85%A9%E1%84%85%E1%85%B2%20%E1%84%89%E1%85%AE%E1%84%8C%E1%85%A5%E1%86%BC%E1%84%87%E1%85%A1%E1%86%BC%E1%84%87%E1%85%A5%E1%86%B8%20c6b07eaf9a534cdcb3a06ffbb6965ba8/%EC%8A%A4%ED%81%AC%EB%A6%B0%EC%83%B7_2022-01-01_20.13.34.png)

고쳐야 하는 내 성격상 별 일 안일어나고, 오류코드를 검색해도 안나오고.. published가 잘만 뜨길래 별 상관 안했으나.. 

utterance가 안뜨는 문제가 지속적으로 발생해서 혹시나 이 에러때문인가 하여 수정하기로 한다.

에러 내용을 읽어보면 

node:46998 [DEP0148] 디프리케이션 워닝이라고 말하며, 아래 경로의 파일에서 ./ 을 ./*으로 바꿔주라고 한다.

아마 노드 버전이 올라가면서 “./” 이것과 같은 문법을 지원하지 않는가 보다.

두가지 방법이 있을것이다. 노드 버전을 예제와 같은 버전으로 낮추던지, 아님 아래 경로에 들어가서 바꿔주던지.

난 후자의 방법을 택하기로 한다. 괜히 노드 버전 낮췄다가 새롭게 적용한 기능들이 작동안할수도 있고, 노드 버전때문에 이미 두통을 많이 겪었던 터라..

게츠비 테마를 클론한 폴더에서 아래 경로로 이동한다.

```bash
클론한 폴더/node_modules/postcss-js/package.json
```

package.json 파일에서 ctrl + f 를 이용해 “./” 를 찾아준다.

![스크린샷 2022-01-01 20.27.59.png](node%2046998%20%5BDEP0148%5D%20%E1%84%8B%E1%85%A9%E1%84%85%E1%85%B2%20%E1%84%89%E1%85%AE%E1%84%8C%E1%85%A5%E1%86%BC%E1%84%87%E1%85%A1%E1%86%BC%E1%84%87%E1%85%A5%E1%86%B8%20c6b07eaf9a534cdcb3a06ffbb6965ba8/%EC%8A%A4%ED%81%AC%EB%A6%B0%EC%83%B7_2022-01-01_20.27.59.png)

```bash
"./":"./" 이렇게 되어있는걸
"./":"./*" 이렇게 바꿔준다.
```

저장 후 다시 nom run deploy 해준다.

에러가 안난다.

![스크린샷 2022-01-01 21.01.15.png](node%2046998%20%5BDEP0148%5D%20%E1%84%8B%E1%85%A9%E1%84%85%E1%85%B2%20%E1%84%89%E1%85%AE%E1%84%8C%E1%85%A5%E1%86%BC%E1%84%87%E1%85%A1%E1%86%BC%E1%84%87%E1%85%A5%E1%86%B8%20c6b07eaf9a534cdcb3a06ffbb6965ba8/%EC%8A%A4%ED%81%AC%EB%A6%B0%EC%83%B7_2022-01-01_21.01.15.png)
